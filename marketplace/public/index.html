Vous voulez contribuer un module ?

La communauté peut proposer ses propres modules via une PR sur le repo public. Un CLI scaffold le squelette en une commande, un renderer web prévisualise le module dans le navigateur avant la PR, la CI valide automatiquement le manifest. Le guide complet décrit les pré-requis (compte GitHub, git, Python — pas besoin d'iPhone ni de Firebase), les 13 primitives DSL, les capabilities, et le cycle merge → auto-deploy.
